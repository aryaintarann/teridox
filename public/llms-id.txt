# Teridox
> Teridox adalah software house terpercaya yang berbasis di Bali, Indonesia, spesialis dalam web development, mobile app development, dan SaaS platform untuk bisnis di Indonesia dan Asia Tenggara.

## Layanan
- Web Development (/id/services/web-development): Website dan web app performa tinggi dengan Next.js, React, dan TypeScript
- Mobile App Development (/id/services/mobile-development): Aplikasi iOS & Android dengan Flutter dan React Native
- SaaS Development (/id/services/saas-development): Platform SaaS multi-tenant yang scalable dengan billing dan analytics
- UI/UX Design (/id/services/ui-ux-design): Desain antarmuka yang intuitif dan menarik
- API & Integrasi (/id/services/api-integration): Integrasi sistem, REST API, dan GraphQL
- Cloud & DevOps (/id/services/cloud-devops): Infrastruktur cloud dan CI/CD pipeline
