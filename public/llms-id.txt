# Teridox
> Teridox adalah full-service digital agency yang berbasis di Bali, Indonesia, spesialis dalam web development, mobile app development, UI/UX design, integrasi AI, dan digital marketing untuk bisnis di Indonesia dan Asia Tenggara. Beroperasi sejak 2024.

## Layanan
- Web Development (/id/services/web-development): Website dan web app performa tinggi dengan Next.js, React, dan TypeScript
- Mobile App Development (/id/services/mobile-development): Aplikasi iOS & Android native menggunakan Flutter dan React Native
- UI/UX Design (/id/services/ui-ux-design): Desain antarmuka intuitif yang berfokus pada konversi
- Integrasi AI (/id/services/ai-integration): Fitur berbasis AI, chatbot, dan otomasi menggunakan LLM
- Software/SaaS Development (/id/services/saas-development): Platform SaaS multi-tenant yang scalable
- Digital Marketing (/id/services/digital-marketing): SEO, media sosial, dan strategi pertumbuhan bisnis

## Tech Stack
- Frontend: Next.js 16, React, TypeScript, Tailwind CSS
- Mobile: Flutter, React Native
- Backend: Node.js, Supabase (PostgreSQL)
- AI: NVIDIA NIM (LLaMA 3.1), Claude API
- Cloud: Vercel, Supabase
- CMS: Admin panel custom yang dibangun sendiri

Semua proyek dibangun dengan dukungan bilingual (Indonesia + Inggris) sebagai standar.
