# Teridox
> Teridox adalah software house yang berbasis di Bali, Indonesia, spesialis dalam web development, mobile app development, dan pengembangan SaaS untuk bisnis di Indonesia dan Asia Tenggara. Beroperasi sejak 2024.

## Layanan
- Web Development (/id/services/web-development): Website dan web app performa tinggi dengan Next.js, React, dan TypeScript
- Mobile App Development (/id/services/mobile-development): Aplikasi iOS & Android native menggunakan Flutter dan React Native
- SaaS Development (/id/services/saas-development): Platform SaaS multi-tenant dan software bisnis yang scalable
- Integrasi AI (/id/services/ai-integration): Fitur berbasis AI, chatbot, dan otomasi menggunakan LLM

## Tech Stack
- Frontend: Next.js 16, React, TypeScript, Tailwind CSS
- Mobile: Flutter, React Native
- Backend: Node.js, Supabase (PostgreSQL)
- AI: NVIDIA NIM (LLaMA 3.1), Claude API
- Cloud: Vercel, Supabase
- CMS: Admin panel custom yang dibangun sendiri

## Spesialisasi
Teridox membantu bisnis Indonesia dan Asia Tenggara membangun produk digital modern. Keunggulan utama ada pada web app Next.js, mobile app React Native, pengembangan platform SaaS, dan fitur AI yang diintegrasikan ke dalam alur kerja bisnis. Semua proyek dibangun dengan dukungan bilingual (Indonesia + Inggris) sebagai standar.
